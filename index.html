Presentation behavior can be fine-tuned using a wide array of configuration options. These objects can be included where you initialize reveal.js. It's also possible to change config values at runtime. Reveal.js can automatically animate elements across slides. All you need to do is add data-auto-animate to two adjacent slide elements and Auto-Animate will animate all matching elements between the two.
